# Outlook UI Clone - React Application

A React application that replicates the Outlook sidebar UI with full routing functionality.

## Features

- Outlook-style sidebar with navigation icons
- Expandable/collapsible email accounts
- Folder structure (Inbox, Drafts, Archive, etc.)
- Unread email counts
- Modern, responsive design
- Full page routing (Mail, Calendar, People, Tasks, Apps)
- Each page has unique UI and functionality

## Installation

1. Install dependencies:
```bash
npm install
```

## Running the Application

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## Build for Production

```bash
npm run build
```
